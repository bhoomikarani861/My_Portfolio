# Professional Developer Portfolio

A state-of-the-art React portfolio featuring modern design trends like Liquid Glassmorphism, deep matte dark mode, and smooth animations.

## ✨ Features

- **Liquid Glass UI**: Sophisticated glassmorphism using `backdrop-filter`.
- **Deep Matte Dark Mode**: Premium dark aesthetic with neon cyan and purple accents.
- **Bento Grid Layout**: Modern project showcase.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Animations**: Silky smooth transitions using Framer Motion.
- **SEO Optimized**: Meta tags and semantic HTML included.

## 🚀 Getting Started

Follow these steps to run the project locally:

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components/`: Reusable React components (Hero, Navbar, Projects, etc.)
- `src/index.css`: Global styles and 'Liquid Glass' theme tokens.
- `public/`: Static assets.

## 🎨 Customization

- **Colors**: Update CSS variables in `src/index.css` under `:root`.
- **Content**: Modify project data in `src/components/Projects.jsx` and skill categories in `src/components/Skills.jsx`.
